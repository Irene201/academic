Om CSS
Mall för CSS
  1. Vad är CSS?
    • Varför finns CSS?
      • Det gör webbplatser grafiskt tilltalande
      • Det är smidigt att bygga upp layouter
      • Det gör webbplatser lättare att underhålla
    • Hur använder vi oss av CSS?
      • CSS är uppbyggt av olika egenskaper som kan ha olika värde
      • Typisk användning av CSS är för typografi, layout, bildhantering, animationer m.m.
      • Genom att koppla olika CSS-mallar till samma HTML-dokument kan informationen presenteras på olika sätt.
      • Anpassa webbplatsen efter målgruppen
      • Öka användbarheten
      • Öka attraktivitete
      • Optimera för användaren
  2. Det finns olika versioner av CSS:
    • Vilka versioner finns det?
      • Nivå 1: Ursprung rekommendation från 1996
      • Nivå 2 (2.1): 2007, primär version med bra webbläsarstöd
      • Nivå 3: Under aktiv utveckling.
    • Varför finns det olika versioner? (Varför är det inte bara en?)
      • CSS har utvecklats genom tiden
    • Varför är det viktigt att veta vilken version som en viss CSS-egenskap tillhör? (Kan detta påverka webbplatsen?)
      • Det är bara de nya webbläsarna som stödjer version 3. Alla moderna webbläsare förutom Internet Explorer stödjer version 2.
      • Om inte webbläsaren stödjer versionen så fungerar detta inte på webbplatsen. Detta bör man vara medveten om om en användare använder en äldre webbläsare.
